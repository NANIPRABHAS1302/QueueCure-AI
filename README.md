# QueueCure AI

AI-Powered Hospital Queue Management System

## Features

* Patient Registration
* Queue Management
* Token Generation
* Doctor Management
* Appointment Scheduling
* Real-Time Queue Updates
* JWT Authentication
* MongoDB Database
* React Frontend Dashboard

## Tech Stack

### Frontend

* React
* Vite
* JavaScript

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* Socket.IO

## Installation

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Environment Variables

Frontend:

```env
VITE_API_URL=http://localhost:5000
```

Backend:

```env
PORT=5000
MONGODB_URI=mongodb://127.0.0.1:27017/queuecure
JWT_SECRET=your_secret_key
```
